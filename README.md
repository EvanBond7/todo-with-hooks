# Todo App

![Интерфейс Приложения](/media/todo-app.gif)

## Технологии

- React
- React Hooks
- Bootstrap

## Основные функции

1.  Создание задач
2.  Возможность редактирования
3.  Отслеживание выполненных задач

## Запуск приложения

```bash
npm start
```

Откройте [http://localhost:3000](http://localhost:3000) с помощью любого браузера
